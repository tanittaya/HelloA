1. **ดําวน์โหลด Git:** ไปที่ [เว็บไซต์ git-scm.com](https://git-
scm.com/downloads) และดําวน์โหลด Git ส ําหรับระบบปฏิบัติกําร (Windows,

macOS, Linux)
